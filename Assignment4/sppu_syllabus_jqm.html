Select Syllabus Year

Semester V
  • Web Technologies: HTML, CSS, JavaScript, PHP, AJAX, introduction to Node.js.
  • Software Engineering & Project Management: SDLC models, Agile, risk and quality management.
  • Artificial Intelligence: Problem solving, search strategies, knowledge representation.
  • Database Management Systems: ER modeling, normalization, transactions, indexing.
  • Elective I (e.g. Data Mining): Classification, clustering, association rule mining.
  • Laboratories: Web Tech Lab, DBMS Lab, AI Lab, Mini Project – I.
Semester VI
  • Cloud Computing: Virtualization, cloud models, AWS & Azure introduction.
  • Information and Cyber Security: Cryptography, network security, attacks and defenses.
  • Machine Learning: Supervised, unsupervised learning, regression, decision trees.
  • Internet of Things (IoT): Architecture, sensors, protocols, applications.
  • Elective II (e.g. Blockchain Technology): Distributed ledger, consensus algorithms, smart contracts.
  • Laboratories: Cloud Lab, Security Lab, ML Lab, Mini Project – II.
Semester V
  • Operating Systems: Processes, threads, scheduling, memory and file systems.
  • Computer Networks: OSI model, TCP/IP, routing, switching, network security basics.
  • Object-Oriented Programming: C++, classes, inheritance, polymorphism, STL.
  • Theory of Computation: Finite automata, regular languages, CFGs, Turing machines.
  • Elective I (e.g. Data Science): Python for data analysis, statistics, visualization tools.
  • Labs: OS Lab, CN Lab, OOP Lab, Mini Project – I.
Semester VI
  • Compiler Design: Lexical analysis, parsing, syntax trees, code generation.
  • Embedded Systems: Microcontrollers, I/O interfacing, real-time operating systems.
  • Advanced Databases: NoSQL, transactions, concurrency, distributed databases.
  • Elective II (e.g. AR/VR): Graphics pipeline, sensors, Unity basics, AR tools.
  • Project Lab: Major Project Phase I.
Semester V
  • Software Modeling and Design: UML diagrams, design patterns, architecture design.
  • Mobile Computing: Android/iOS app development, mobile networks, sensors.
  • Operating System Internals: Kernel, memory management, inter-process communication.
  • Open Elective I (e.g. Soft Computing): Fuzzy logic, neural networks, genetic algorithms.
  • Laboratories: Mobile App Lab, Software Modeling Lab, Mini Project – I.
Semester VI
  • Digital Signal Processing: DFT, FFT, digital filters, image and speech processing.
  • Wireless Communication: Antennas, modulation techniques, Bluetooth, Wi-Fi.
  • Data Visualization: Tableau, Power BI, D3.js for interactive visual dashboards.
  • Open Elective II (e.g. Human-Computer Interaction): Usability, UI/UX design, accessibility.
  • Project Lab: Major Project Phase I.
